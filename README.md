# Parawa — prototipo clickeable

Pantallas navegables con **Next.js + shadcn/ui** y datos de ejemplo. Sin backend.

## Pantallas

| Ruta | Descripción |
|------|-------------|
| `/` | Landing (marketing) |
| `/admin` | Admin dashboard |
| `/admin/providers` | Gestión proveedores |
| `/admin/bookings` | Todas las reservas |
| `/admin/users` | Usuarios |
| `/admin/verifications` | Cola de verificación |
| `/discover` | Explorar proveedores |
| `/providers/[id]` | Perfil + reservar + mensaje |
| `/bookings` | Mis reservas |
| `/provider` | Panel proveedor |
| `/messages` | Lista de chats |
| `/messages/[id]` | Chat |
| `/login` | Entrar (enlaces demo) |

## Ejecutar

```bash
cd next-app
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).
